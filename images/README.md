# Screenshots Folder

This folder is for the screenshots described in `docs/screenshots_checklist.md` — RTL code, testbench, waveform views, Vivado synthesis/utilization reports, and (optionally) a physical FPGA board photo/video.

Add your `.png`/`.jpg` files here before publishing the repository, then reference them in the README (e.g. `![Waveform](images/02_alarm_waveform.png)`).
